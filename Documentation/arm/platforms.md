# Novusk - ARM platforms

Last edited: 2022/11/5

---

This is a list of devices Novusk supports and the values for the ``PLATFORM`` argument when building.

### ARM 32 devices

- RaspberryPi 2, ``rpi2``
- LM3S6965 (Stellaris 6965), ``stellaris6965``

### ARM 64 devices

- RaspberryPi 3, ``rpi3``
- Qemu Virt, ``virt``
